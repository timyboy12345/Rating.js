Review.js

This is the examples page for Review.js. Look at the examples below to get an idea of what this module is capable of.

These examples use Font Awesome for the icons, but you're free to use any icon you wish by changing the icon classes in the options.

You can look at the source code in the developer console to view the code used to create these examples.
